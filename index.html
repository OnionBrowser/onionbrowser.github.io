Enhance your web privacy

Onion Browser is a free and open-source web browser for iPhone and iPad that encrypts and tunnels web traffic through the Tor network, with extra features to help you browse the internet privately. See more features & benefits.

  • Onion Browser is recommended by Tactical Technology Collective and ProPublica’s Guide to Safer Communication.
  • Featured in Ars Technica, The New York Times, Salon, TechCrunch, Boing Boing, Gizmodo, Lifehacker, The Guardian, Cult of Mac, and others.
  • Read about Onion Browser on the official Tor Project blog.

Latest version:

  • Onion Browser 2.3.0 (December 2019) - Native dark mode support, fix for OCSP security issue, ability to download files & share with other apps, redesigned settings page, and more. Users are encouraged to update via App Store.

Onion Browser is a free app. If you'd like to support continuing development of Onion Browser and other Tor iOS projects, please consider sending a tip or other financial contribution. Even a couple dollars helps!

Features & Benefits

  • Internet access is tunneled through the Tor network[1][2]
    • Websites do not see your real IP address.
    • ISPs and insecure wireless networks cannot see your browsing.
    • Access websites, even behind some types of internet filters and censors.
    • View .onion websites: the "dark net" of hidden sites only accessible using Tor.
  • Block third party cookies or all cookies.
  • Ability to change cookie storage policy (Allow All / Block Third Party / Block All)
  • Disable scripts and multimedia content that can be used to track you.
  • Can send the "Do Not Track" HTTP header (DNT: 1) to websites.

Security Advisories, Known Bugs, and Etc. (Updated 2019-12-17)

See the release history for a log of Onion Browser release notes, including previous security issues and results of previous security audits. See the Github project for a current list of known issues.

Using Onion Browser does not guarantee security or privacy by itself. It simply provides a set of features that may enhance privacy and anonymity while browsing the web. To stay anonymous, you should also consider your behavior as you use Onion Browser or other privacy-protecting software.

  • Onion Browser only tunnels traffic within the Onion Browser app. Information from other apps (and iOS system traffic) will continue to use your normal internet connection.
  • Advisory: WebRTC/Media leaks. Due to iOS limitations, WebRTC and media files leak outside of Tor and are routed over the normal internet. This will reveal your real IP address to sites using these features. (If you are using a VPN, the VPN IP address is revealed instead.) To defend against this, you may set Strict security mode in Host Settings, which will disable Javascript. More information here.
  • Advisory: Time Zone and other basic system information: Websites may use Javascript to ask the browser for local time, therefore revealing your real time zone. Websites may also try to use the HTML5 Geolocation API (to view your current GPS location) or other new Javascript features to reveal information from iOS. Disabling Javascript using the Strict security setting, and being aware of any permissions pop-ups, are the only way to bypass this.

Privacy Policy & Privacy Notices

Onion Browser does not directly collect any data on user activity. Onion Browser does not process any user data. Other than what is built into iOS, Onion Browser does not utilize any third party analytics.

Information on iOS data collection (App Store account info, opt-in analytics, etc), and its privacy policy (and full legal compliance information), can be found here: https://www.apple.com/legal/privacy/en-ww/

iOS analytics data collection is opt-in (Settings->Privacy->Analytics in iOS) and is aggregated and anonymized using differential privacy techniques to make it difficult or impossible to extract an individual user’s information from the aggregated data. https://images.apple.com/privacy/docs/Differential_Privacy_Overview.pdf

Apple only makes this data available in aggregate and the Onion Browser developers do not have access to an individual user’s information. For examples of the aggregate data made available, see: https://developer.apple.com/app-store/app-analytics/
